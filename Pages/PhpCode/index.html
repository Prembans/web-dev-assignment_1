Understanding PHP

What is PHP?

PHP (Hypertext Preprocessor) is a widely-used, open-source server-side scripting language designed specifically for web development.

Unlike HTML, CSS, and JavaScript—which are executed in the user's web browser (client-side)—PHP code is executed on the web server. The server processes the PHP code and then sends pure, readable HTML back to the user's browser.

What can PHP do?

  • Dynamic Content: It can generate dynamic page content.
  • Form Handling: It can collect data from web forms.
  • Database Integration: PHP is heavily used to add, delete, and modify data in databases (like MySQL).
  • Security: It can encrypt sensitive data.

10 Essential PHP Concepts

1. How do you print "Hello, World!" to the screen?

<?php echo "Hello, World!"; ?>

2. How do you create variables and add two numbers?

<?php $num1 = 15; $num2 = 25; $sum = $num1 + $num2; echo "The sum is: " . $sum; ?>

3. How do you check if a number is Even or Odd using an If/Else statement?

<?php $number = 8; if ($number % 2 == 0) { echo "$number is Even"; } else { echo "$number is Odd"; } ?>

4. How do you use a 'For' Loop to count from 1 to 5?

<?php for ($i = 1; $i <= 5; $i++) { echo "The number is: $i <br>"; } ?>

5. How do you create an Array and loop through it?

<?php $movies = array("Avengers", "Godzilla", "Dune"); foreach ($movies as $movie) { echo $movie . "<br>"; } ?>

6. How do you create a custom Function to calculate a factorial?

<?php function factorial($n) { if ($n <= 1) { return 1; } else { return $n * factorial($n - 1); } } echo "Factorial of 5 is: " . factorial(5); ?>

7. How do you handle form data submitted via POST?

<?php if ($_SERVER["REQUEST_METHOD"] == "POST") { $name = htmlspecialchars($_POST['fname']); if (empty($name)) { echo "Name is empty"; } else { echo "Welcome, " . $name; } } ?>

8. How do you start a Session and set a Session variable?

<?php session_start(); $_SESSION["username"] = "AdminUser"; $_SESSION["role"] = "Administrator"; echo "Session variables are set."; ?>

9. How do you safely read the contents of a file?

<?php $filename = "webdictionary.txt"; $myfile = fopen($filename, "r") or die("Unable to open file!"); while(!feof($myfile)) { echo fgets($myfile) . "<br>"; } fclose($myfile); ?>

10. How do you connect to a MySQL Database using PDO?

<?php $servername = "localhost"; $username = "root"; $password = "password"; try { $conn = new PDO("mysql:host=$servername;dbname=myDB", $username, $password); $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION); echo "Connected successfully"; } catch(PDOException $e) { echo "Connection failed: " . $e->getMessage(); } ?>
